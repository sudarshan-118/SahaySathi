## 📌 Overview

**SahaySathi** is a modern web platform built to provide smart and accessible solutions for sustainability-focused community problems. The application is designed with a clean user experience, scalable architecture, and responsive interface to ensure accessibility across devices.

The project focuses on combining technology with social impact, helping users interact with sustainable initiatives efficiently through a fast and intuitive platform.

* ⚡ Fast and optimized performance using **Next.js**

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/sudarshan-118/SahaySathi.git
```

### 2️⃣ Navigate into the Project

```bash
cd SahaySathi
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run the Development Server

```bash
npm run dev
```

Now open:

```bash
http://localhost:3000
```
